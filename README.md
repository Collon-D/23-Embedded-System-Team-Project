## ⚙️ 실행 방법
- 컴파일
```bash
$ make
```
- 실행
```bash
$ ./Alarm
```

## 📌 기능
### 블루투스 연결

- 양식에 맞는 값이 들어오면 해당 알람이 실행된다.
